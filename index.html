An Electron based Docker Swarm Manager

Electroswarm v0.0.1 (in development)

Electroswarm is a desktop, Electron-based application to manage Docker Swarms.

  • Visualize services and nodes in your Swarm
  • Scale services down and up
  • Run simple SSH commands agains the Swarm's manager
  • Tested with
    • Docker Toolbox, version 1.12.0, build 8eab29e
    • Windows 10 Professional, x64
